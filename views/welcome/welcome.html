Natural

VeloxJS try to be as natural as possible. Views is HTML file. Styling is CSS. Scripting is vanilla Javascript
Behaviour should be as intuitive as possible with as least cryptic magic as possible

Complete and modular

VeloxJS is complete with view, controller, database handling, offline sync and many high level features
But it is modular and each part can be used on its own. You like VeloxJS views but don't need the other stuff ? just load the view module and use it alone

Light and performant

VeloxJS is light. Really. The VeloxJS view (min+gzip) is 8.4KB. The whole VeloxJS view + all extensions is 21.3KB.
Performance of VeloxJS does not rely on a magic new render engine but in the fact that it does few magic and keep its technical concept as close as what the browser is design to do as possible
