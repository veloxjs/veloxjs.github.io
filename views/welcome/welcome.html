What is VeloxJS ?

VeloxJS aims to be both a full JS framework and a set of independents tools. You can use it as a framework to get a full software architecture or you can only take the parts that you want in its features.

You can get more details in my full explaination about why I wrote a JS framework

Demo

The usual TODO app demo demonstrate a some interresting feature of VeloxJS :

  • The welcome view do some real world application stuf such as :
    • 2 level nested repeating subviews for projects and task
    • CSS styling rules
    • Custom JS to handle drag&drop
    • Popup for quick task creation
  • The main view with its menu and language switcher
  • The task view with a classical CRUD grid/form UI
